# 🚀 TypeBoi

> iOS-based smart utility app for OCR, translation, and AI-powered text tools.

---

## ✨ Overview

TypeBoi is a productivity app designed to make text interaction seamless on mobile.

It combines:

- 📸 OCR (Image → Text)
- 🌍 Translation
- 🧠 AI-based text processing
- 🎤 Speech recognition & TTS

All in one lightweight iOS app.

---

## 🔥 Key Features

### 1. OCR Text Capture
- Extract text from images instantly
- Built using iOS Vision framework

### 2. Smart Translation
- Real-time translation
- Multi-language support

### 3. AI Text Processing
- Grammar correction
- Rewrite / improve sentences
- Context-aware suggestions

### 4. Voice Integration
- Speech → Text
- Text → Speech

---

## 🧱 Tech Stack

- Swift / SwiftUI
- iOS Vision Framework (OCR)
- Speech Framework (STT / TTS)
- OpenAI API (optional AI features)
- Supabase (optional backend)
